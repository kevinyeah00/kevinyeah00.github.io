博客的诞生

这个博客是怎么诞生的？

前期

通过ceba的介绍，来到 hexo ，安装了必要的程序 node 和 git ,然后发现b站上有一些教学，便跟着教学完成了博客。

b站教学链接：https://www.bilibili.com/video/av44544186?from=search&seid=15660159090124681135

遇到的问题与解决

  • 个别步骤速度过慢 ———— 重新开始，直接在git里面打开即可。
  • 在进入github时过慢 ———— 修改hosts文件，但是记着以后不要乱改管理员权限，手动狗头。

注意事项

  • 素质三连

  • _config 文件很重要

存在的问题

  • 由于wall的原因，没有办法愉快更换主题，因此下一步的任务就是改造我那个只支持ipv6的梯子，再次手动狗头。
